EL INTERNET COMO MEDIO DE COMUNICACIÓN


Funcionamiento del Internet

El Internet opera como una red global interconectada que usa protocolos como TCP/IP. Los datos se dividen en paquetes que viajan por routers y cables (como la fibra óptica) y se reensamblan en el destino. El sistema de nombres de dominio (DNS) se usa para buscar direcciones IP de servidores.

  • Usos como Medio de Comunicación:
    • Acceso a la información: Permite el acceso instantáneo a noticias y datos educativos.
    • Comunicación interpersonal: Facilita el contacto directo por correo electrónico, redes sociales y videollamadas.
    • Medio masivo: Sirve como plataforma para difundir contenido multimedia e información a gran escala a través de blogs y sitios de noticias.
    • Herramienta educativa: Se utiliza para la enseñanza y el acceso a recursos en línea.
